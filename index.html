Traffico provides a set of layers that can be composed into traffic signs icons in a form of a web font. Accompanying .json files allow to generate templates on the fly. Check out Shapes page for available shapes or Examples for what is available in the .json files.

Install from command line: $ bower install traffico

Generative templates

With every Traffico release, there is a set of .json files attached, which provide for a good foundation for generating templates in a framework of your choice.
